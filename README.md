## About
Devboard provides you with an interface to connect your GitHub and view your repo metrics. You can connect multiple repos and view pulls repository data, and shows engineering health metrics — PR cycle time, deployment frequency, code churn, contributor activity — in a clean, interactive dashboard.

## Features
- Secure session based authentication
- Connect your GitHub securely via OAuth
- Interactive dashboard to view all your repos
- Track/Untrack repos by choice
- View individual repo metrics
- Used Recharts to create interactive charts
- Type safe APIs using TRPC
- Fetching and caching data using Tanstack
